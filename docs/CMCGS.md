# Continous Monte Carlo Graph Search

<div align="center">

<img src="../examples/gifs/cmcgs_kbf.gif" width="200" alt="CGA Example">

<p><b>Figure:</b> Continous Monte Carlo Graph Search (CMCGS) is an adaption of Monte Carlo Tree Search to continuous problem spaces. Rather than operating with a search tree, (which would require an action space of infinite measure in continuous domains), CMCGS transforms the search tree into a layered graph, using clustering to bound the size of the action space.</p>
